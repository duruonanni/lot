## Add a Tool

Use this workflow when adding a new AI tool to the directory.

### 1. Gather inputs

| Field | Required | Example |
|-------|----------|---------|
| `id` | yes | `invoice-extractor` (kebab-case, unique) |
| `name.en` / `name.zh` | yes | Lenovo EaaS Invoice Validator |
| `summary.en` / `summary.zh` | yes | 1–2 sentence description |
| `url` | yes | `https://…` |
| `category` | yes | `validation` \| `extraction` \| `analysis` \| `automation` \| `other` |
| `status` | yes | `pilot` \| `ga` \| `deprecated` |
| `tags` | no | `["pdf", "validation"]` |
| `owner` | no | Default `DT` |
| `docsUrl` | no | Optional documentation link |

### 2. Edit `data/tools.json`

Append a new object to the `tools` array and update `meta.lastUpdated` to today's date (ISO `YYYY-MM-DD`).

### 3. Preview locally

```bash
npm run preview
```

Confirm the card renders in both English and Chinese.

### 4. Publish

Commit and push to `main`. Netlify redeploys automatically.
